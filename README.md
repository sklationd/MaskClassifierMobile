# MaskClassifierMobile
Mobile version of mask classifier, based on [this repo](https://github.com/pytorch/android-demo-app/tree/master/PyTorchDemoApp).

## Environment
```
pytorch-android == '1.7.0'
torchvision-android == '1.7.0'
```
